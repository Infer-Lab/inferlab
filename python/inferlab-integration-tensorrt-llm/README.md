# Inferlab Integration for TensorRT-LLM

Framework-specific planning and rendering for running TensorRT-LLM servers
through Inferlab. The consuming workspace supplies TensorRT-LLM and its
hardware runtime; this package supplies only the Inferlab integration boundary.

See the [Inferlab repository](https://github.com/Infer-Lab/inferlab) for
workspace authoring and supported topology documentation.
