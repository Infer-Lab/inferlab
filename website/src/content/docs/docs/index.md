InferLab documentation is organized around the operator workflow and the repository artifacts that own each fact.

- **Getting Started** locates the release installation and first workspace commands.
- **Concepts** explains how to read the workspace, local bindings, and records model without replacing the normative specification.
- **Guides** project the maintained workspace-authoring and view-only TUI guides.
- **Reference** projects the qualified backend support matrix.
- **Architecture & Specification** publishes the current RFC and ADR corpus from the filtered public snapshot.

Start with [installation and quick start](./getting-started/installation/) or go directly to the [specification authority map](./architecture/rfc/rfc-0001/).
